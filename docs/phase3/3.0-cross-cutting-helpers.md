# 3.0 — สูตรกลาง (helper) ที่หลายหน้าใช้ร่วมกัน

> **3.0 ไม่มีหน้าเว็บโดยตรง** — ตรวจสูตรกลางที่ใช้ใน 12+ endpoint ถ้าตรงนี้ผิด ทุกหน้าจะเห็นตัวเลขผิดตาม

## หน้าเว็บที่ตรวจ

- **ชื่อหน้า:** ทุกหน้าที่ใช้สูตรเหล่านี้ (รายละเอียดในตาราง "Bugs ที่พบ")
- **URL:** ครอบคลุม — Executive, Operations, Threat Intelligence, IOC pages ทั้งหมด
