| # | ประเด็น | ผลต่อหน้าเว็บ |

| 8 | `recent_stats` parameter ของ `_operations_overview_from_aggs` ไม่ได้ใช้แล้วหลัง fix | คงไว้ใน signature เพื่อ backward-compat — ลบใน follow-up cleanup ได้ |
