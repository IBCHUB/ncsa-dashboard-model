### Auth flow

| Endpoint | Public? | สูตร | ถูกไหม |
|----------|---------|------|--------|
| `POST /auth/login` | Public | username/password → set HttpOnly cookie + return token | ✅ Phase 2.8 fix (HttpOnly=true) |
| `GET /auth/me` | Bearer | return current user public view | ✅ |
| `POST /auth/logout` | Bearer | invalidate token | ✅ |
